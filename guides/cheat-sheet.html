  • 1. Git Command Cheat Sheet 
    Cheat Sheet

Update main
git checkout main
git pull origin main

New branch
git checkout -b feature/yourname-task

See status
git status

Stage & commit
git add .
git commit -m "feat: describe change"

Push first time
git push -u origin feature/yourname-task

Merge main into branch
git checkout main
git pull origin main
git checkout feature/yourname-task
git merge main

Resolve conflicts
git status
# Edit files to remove markers, then:
git add file
git commit

Rebase (optional)
git checkout feature/yourname-task
git rebase main
git rebase --continue

Delete after merge
git checkout main
git branch -d feature/yourname-task
git push origin --delete feature/yourname-task

Notes

  • If your branch is already named main, skip the rename step.
  • You can use VS Code's Source Control view for GUI commits.
  • Install GitHub CLI from cli.github.com.
