Create your own project using Git Commands with VSCode

Learn how to set up Git in VS Code, commit your work, and push your project to GitHub â€” all from the terminal.

  • 1. Open VS Code

    Open VS Code and bring up the terminal by pressing Ctrl + `.

  • 2. Go to your Documents folder
    cd ~/Documents
  • 3. Create a new project folder
    mkdir MyProject cd MyProject
  • 4. Initialize Git
    git init
  • 5. Create your first file
    echo "# MyProject" > README.md
  • 6. Stage and commit your work
    git add . git commit -m "Initial commit"
  • 7. Create a GitHub repository (with GitHub CLI)
    gh repo create MyProject --private --source=. --remote=origin # or make it public: gh repo create MyProject --public --source=. --remote=origin
  • 8. Push your project to GitHub
    git branch -M main git push -u origin main

Notes

  • If your branch is already named main, skip the rename step.
  • You can use VS Code's Source Control view for GUI commits.
  • Install GitHub CLI from cli.github.com.
